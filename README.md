# Hermit C2

Command & Control, Post-Exploitation Framework written in Rust.  
I developed this project for learning Rust and how C2 framework works.

<br />

## USAGE

### C2 Server

```sh
hermit server
```

### C2 Client

```sh
hermit client

# Print the usage.
hermit > help
```

<br />

## Installation

```sh
git clone https://github.com/hideckies/hermit.git
cd hermit
cargo install
cargo build
```
